## Repositories

### Frontend (This Repository)

- Built using React, Vite, and Tailwind CSS  
- Integrated with Google OAuth, smart search, filtering, and AI fallback recipe generation

### Backend Repository

https://github.com/AdarshKumarSr/Smart-Recipe-Generator-

- Spring Boot, MongoDB, Gemini AI, JWT authentication, and Docker  
- Includes service layer logic, filtering, scoring, and seed data  
- Full documentation available in the repository

## Features

- Ingredient-based recipe search  
- AI-generated suggestions when no suitable match exists  
- Filtering by diet, cuisine, difficulty, time, and tags  
- Google OAuth with JWT authentication  
- Deployment: Render (frontend) and Railway (backend)
